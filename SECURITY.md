# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Cathartic, please report it privately through GitHub's Security Advisory system:

1. Go to https://github.com/20player11/cathartic/security/advisories
2. Click "Report a vulnerability"
3. Fill in the details

Do **not** report security vulnerabilities via public issues or discussions.

## What to expect

- Acknowledgment within 48 hours
- An assessment and fix timeline within 5 business days
- A CVE assignment if applicable

## Scope

- The Cathartic application itself
- Build and packaging scripts

Out of scope: third-party dependencies (report to respective maintainers).
